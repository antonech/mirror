Mirror is a compile-time template metaprogramming-based reflection library built on top of the basic C++ reflection facility.

While the reflexpr operator and the related library of metaobject operation templates are proposed for standardisation, the Mirror library at this point is not.

Metaobjects

Unlike the basic C++ reflection facility which defines several different metaobject concepts, the Mirror library distinguishes only metaobjects implemented by the metaobject class template and metaobject sequences implemented by metaobject_sequence.

Whenever a metaobject operation is invoked on a metaobject for which it is not applicable a suitable default value or the none type is returned.

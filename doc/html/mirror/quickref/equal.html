operation equal

#include <mirror/equal.hpp>

The equal operation returns true if the pair of basic_strings, ranges or int_consts passed as arguments are equal, false otherwise, as shown in the following pseudocode. 

template <typename Char, Char ... C1, Char ... C2>
using equal<basic_string<Char, C1...>, basic_string<Char, C2...>>
 = bool_<unspecified >;

template <typename ... P1, typename ... P2>
using equal<range<P1...>, range<P2...>>
 = bool_<unspecified >;

true if the strings are equal, false otherwise.

true if the ranges are equal, false otherwise.

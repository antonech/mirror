operation get_reflected_type

#include <mirror/get_reflected_type.hpp>

The get_reflected_type operation returns the base-level type reflected by the argument, as shown on the following pseudocode. 

template <std::meta::Type T>
using get_reflected_type<metaobject<T>>
 = std::meta::get_reflected_type_t<T>;

template <std::meta::Object O>
using get_reflected_type<metaobject<O>> = none ;

using get_reflected_type<none> = none;

For metaobjects which do not reflect a type none is returned.

Note

If the argument is not a wrapped std::meta::Type then the result is is none.

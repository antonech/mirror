operation is_enum

#include <mirror/is_enum.hpp>

The is_enum operation indicates if the passed argument is a metaobject reflecting an enum, as shown in the following pseudocode. 

template <std::meta::TagType T>
using is_enum<metaobject<T>>
 = bool_<std::meta::is_enum_v<T>>;

template <std::meta::Object O>
using is_enum<metaobject<O>> = false_ ;

using is_enum<none> = false_;

For metaobjects not conforming to the std::meta::TagType concept false_ is returned.

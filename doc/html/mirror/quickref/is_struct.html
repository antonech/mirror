operation is_struct

#include <mirror/is_struct.hpp>

The is_struct operation indicates if the passed argument is a metaobject reflecting a struct, as shown in the following pseudocode. 

template <std::meta::TagType T>
using is_struct<metaobject<T>>
 = bool_<std::meta::is_struct_v<T>>;

template <std::meta::Object O>
using is_struct<metaobject<O>> = false_ ;

using is_struct<none> = false_;

For metaobjects not conforming to the std::meta::TagType concept false_ is returned.

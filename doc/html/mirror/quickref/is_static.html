operation is_static

#include <mirror/is_static.hpp>

The is_static operation indicates if the base-level entity reflected by the argument has static linkage or storage class, as shown in the following pseudocode. 

template <std::meta::Variable V>
using is_static<metaobject<V>>
 = bool_<std::meta::is_static_v<V>>;

template <std::meta::Object O>
using is_static<metaobject<O>> = false_ ;

using is_static<none> = false_;

Metaobjects not conforming to the std::meta::Variable concept are considered non-static.

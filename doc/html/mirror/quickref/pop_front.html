operation pop_front

#include <mirror/pop_front.hpp>

The pop_front operation removes a single type from the front of a range or a single character from the front of a string as shown in the following pseudocode. 

template <typename Char, Char C1, Char ... C2>
using pop_front<basic_string<Char, C1, C2...>>
 = basic_string<Char, C2...>;

template <typename P1, typename ... P2>
using pop_front<range<P1, P2...>> = range<P2...>;

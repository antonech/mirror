operation is_class

#include <mirror/is_class.hpp>

The is_class operation indicates if the passed argument is a metaobject reflecting a class, as shown in the following pseudocode. 

template <std::meta::TagType T>
using is_class<metaobject<T>>
 = bool_<std::meta::is_class_v<T>>;

template <std::meta::Object O>
using is_class<metaobject<O>> = false_ ;

using is_class<none> = false_;

For metaobjects not conforming to the std::meta::TagType concept false_ is returned.

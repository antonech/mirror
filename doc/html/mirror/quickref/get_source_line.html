operation get_source_line

#include <mirror/get_source_line.hpp>

The get_source_line operation returns a compile-time int_const with the line number in the source file where the base-level entity reflected by the argument is declared, as shown in the following pseudocode. 

template <__std_meta_object O>
using get_source_line<metaobject<O>> = unsigned_<unspecified >;

using get_source_line<none> = empty_string;

The line number in the source file where the base-level entity reflected by the argument was declared as returned by std::meta::get_source_line. If the base-level entity does not have an in-source declaration (for example a specifier, native type, etc.) then an zero is returned.

operation get_scope

#include <mirror/get_scope.hpp>

The get_scope operation returns a metaobject reflecting the scope of the base-level entity reflected by the argument, as shown on the following pseudocode. 

template <std::meta::ScopeMember SM>
using get_scope<metaobject<SM>>
 = metaobject<std::meta::get_scope_m<SM>>;

template <std::meta::Object O>
using get_scope<metaobject<O>> = none ;

using get_scope<none> = none;

For metaobjects which do not reflect scope members none is returned.

Note

If the argument is not a wrapped std::meta::ScopeMember then the result is is none.

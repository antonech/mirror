operator MIRRORED

#include <mirror/reflection.hpp>

The metaobjects reflecting various program declarations and other base-level entities can be obtained by invoking the MIRRORED operator.

The result of the MIRRORED operator is an instantiation of the metaobject template wrapping a basic metaobject type.

The MIRRORED operator takes the same arguments as the reflexpr operator.

For example: 

using meta_int = MIRRORED(int);
using meta_std = MIRRORED(std);

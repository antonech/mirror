operation get_aliased

#include <mirror/get_aliased.hpp>

The get_aliased operation returns a metaobject reflecting the base-level entity, aliased by the alias reflected by the argument, as shown on the following pseudocode. 

template <std::meta::Alias A>
using get_aliased<metaobject<A>>
 = metaobject<std::meta::get_aliased_m<A>>;

template <std::meta::Object O>
using get_aliased<metaobject<O>> = none ;

using get_aliased<none> = none;

For metaobjects which do not reflect aliases none is returned.

Note

If the argument is not a wrapped std::meta::Alias then the result is is none.

operation get_base_name

#include <mirror/get_base_name.hpp>

The get_base_name operation return a compile-time string containing the base name of the argument as shown in the following pseudocode. 

template <std::meta::Named N>
using get_base_name<metaobject<N>> = string<unspecified >;

template <std::meta::Object O>
using get_base_name<metaobject<O>> = empty_string ;

using get_base_name<none> = empty_string;

The characters of the base name of the std::meta::Named metaobject as returned by std::meta::get_base_name.

For non-named metaobjects an empty string is returned.

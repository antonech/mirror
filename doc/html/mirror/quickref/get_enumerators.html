operation get_enumerators

#include <mirror/get_enumerators.hpp>

The get_enumerators operation returns a metaobject_sequence containing the metaobjects reflecting the enumerators of the enumerated type reflected by the argument, as shown on the following pseudocode. 

template <std::meta::Enum E>
using get_enumerators<metaobject<E>>
 = metaobject_sequence<std::meta::get_enumerators_m<E>>;

template <std::meta::Object O>
using get_enumerators<metaobject<O>> = none ;

using get_enumerators<none> = none;

For metaobjects which do not reflect an enumeration none is returned.

Note

If the argument is not a wrapped std::meta::Enum then the result is is none.

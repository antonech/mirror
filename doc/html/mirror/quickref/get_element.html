operation get_element

#include <mirror/get_element.hpp>

The get_element operation returns the I-th element in a a metaobject_sequence, range, string or none as shown in the following pseudocode. 

template <std::meta::ObjectSequence S, size_t I>
using get_element<metaobject_sequence<S>, size_const<I>>
 = metaobject<std::meta::get_element_m<S, I>>;

template <typename Char, Char ... C, size_t I>
using get_element<basic_string<Char, C...>, size_const<I>>
 = int_const<Char, unspecified >;

template <typename ... P, size_t I>
using get_element<range<P...>, size_const<I>>
 = unspecified ;

template <size_t I>
using get_element<none, size_const<I>> = none;

The I-th character in the pack C.

The I-th type in the pack P.

Note

If the index passed as the second argument is greater or equal to the number of elements in the sequence, range or string, then the result is none.

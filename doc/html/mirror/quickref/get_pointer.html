operation get_pointer

#include <mirror/get_pointer.hpp>

The get_pointer operation returns a compile-time int_const with the type and value of the pointer to the variable or class data member reflected by the argument, as shown in the following pseudocode. 

template <std::meta::Variable C>
using get_pointer<metaobject<C>> = int_const<unspecified , unspecified >;

template <std::meta::Object O>
using get_pointer<metaobject<O>> = none ;

using get_pointer<none> = none;

The type of the pointer to the variable reflected by the argument. If the argument reflects a class member, then this is a pointer-to-data-member type.

The pointer constant value, as if taken with the & operator.

For metaobjects which do not reflect constant values none is returned.

operation get_base_class

#include <mirror/get_base_class.hpp>

The get_base_class operation returns a metaobject reflecting the base class in a class inheritance reflected by the argument, as shown on the following pseudocode. 

template <std::meta::Inheritance I>
using get_base_class<metaobject<I>>
 = metaobject<std::meta::get_base_class_m<I>>;

template <std::meta::Object O>
using get_base_class<metaobject<O>> = none ;

using get_base_class<none> = none;

For metaobjects which do not reflect class inheritance none is returned.

Note

If the argument is not a wrapped std::meta::Inheritance then the result is is none.

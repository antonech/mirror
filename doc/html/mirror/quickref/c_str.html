operation c_str

The c_str operation returns a const character array, containing a null-terminated c-string from a compile-time basic_string as shown in the following pseudocode: 

template <typename Char, Char ... C>
const Char c_str<basic_string<Char, C...>>[sizeof(C)+1] =  {C...,Char(0)};

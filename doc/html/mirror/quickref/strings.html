Compile-time strings

basic_string

The basic_string class template is used to represent compile-time strings with Char character type. 

template <typename Char, Char ... C>
struct basic_string {
	using type = basic_string;
};

string

Compile-time char strings are represented by the following specialization of the basic_string template: 

template <char ... C>
using string = basic_string<char, C...>;

empty_string 

using empty_string = string<>;

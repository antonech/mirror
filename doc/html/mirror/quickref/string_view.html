operation string_view

#include <mirror/string_view.hpp>

The string operation returns a std::basic_string_view from a compile-time basic_string as shown in the following pseudocode: 

template <typename Char, Char ... C>
constexpr std::string_view string_view<basic_string<Char, C...>> = unspecified;

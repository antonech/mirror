operation is_anonymous

#include <mirror/is_anonymous.hpp>

The is_anonymous operation indicates if the passed argument has a “name” or of it is anonymous, as shown in the following pseudocode. 

template <std::meta::Named N>
using is_anonymous<metaobject<N>>
 = bool_<meta::is_anonumoys_v<N>>;

template <std::meta::Object O>
using is_anonymous<metaobject<O>> = true_ ;

using is_anonymous<none> = true_;

Metaobjects not conforming to the std::meta::Named concept are also considered anonymous.

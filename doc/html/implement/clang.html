Getting started

A fork of the clang frontend for llvm implementing the proposed basic reflection facility is hosted on GitHub on the reflexpr branch.

Getting started

In order to build modified version of clang follow the instructions posted here, but instead of checking out the code from the official repository, do the following: 

$> cd llvm/tools
$> git clone https://github.com/matus-chochlik/clang.git
$> cd clang
$> git checkout reflexpr
$> cd ../../..

Note however that the commits made on the reflexpr branch are occasionally rebased against the official master branch, amended and force-pushed to the forked repository. In order to apply the latest changes made in the remote repository to the local repository, instead of plain 

$> git pull

you need to do the following on the reflexpr branch: 

$> git fetch origin
$> git reset --hard origin/reflexpr

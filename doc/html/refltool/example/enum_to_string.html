Enum to string

This example shows how to traverse all enumerators in an enum type and how to convert enum values to strings.

We need to include the following headers: 

#include <refltool/tie_enumerators.hpp>
#include <refltool/enum_to_string.hpp>
#include <iostream>

We'll demonstrate the functionality on the following enum: 

enum E {
	a = 1 << 0,
	b = 1 << 1,
	c = 1 << 2,
	d = 1 << 3,
	e = 1 << 4,
	f = 1 << 5,
	g = 1 << 6,
	h = 1 << 7
};

The tie_enumerators function returns a std::array containing all enumerators in the specified enum type. The enum_to_string function returns the basic name of an enumerator. 

int main(void)
{
	using namespace refltool;

	for(E e : tie_enumerators<E>())
	{
		std::cout
			<< enum_to_string(e)
			<< " = "
			<< int(e)
			<< std::endl;
	}

	return 0;
}

Running the example produces the following output: 

a = 1
b = 2
c = 4
d = 8
e = 16
f = 32
g = 64
h = 128

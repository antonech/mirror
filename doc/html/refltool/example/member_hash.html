Data member hash

This example shows how to hash all class data members and how to use it to implement specializations of std::hash used by std::unordered_set.

We need to include the following headers: 

#include <refltool/data_member_cmp.hpp>
#include <refltool/hash_data_members.hpp>
#include <unordered_set>
#include <iostream>

We'll demonstrate the functionality on two classes -- Point and Triangle. We use the data_members_equal operation and the hash_data_members function to implement their equality comparison operators and specializations of std::hash: 

struct Point
{
	int x, y, z;
};

bool operator == (const Point& a, const Point& b)
{ return refltool::data_members_equal(a, b); }

namespace std {

template <>
struct hash<Point>
{
	std::size_t operator()(const Point& p) const
	{ return refltool::hash_data_members(p); }
};

} // namespace std

struct Triangle
{
	Point a, b, c;
};

bool operator == (const Triangle& a, const Triangle& b)
{ return refltool::data_members_equal(a, b); }

namespace std {

template <>
struct hash<Triangle>
{
	std::size_t operator()(const Triangle& t) const
	{ return refltool::hash_data_members(t); }
};

} // namespace std

Now we can demonstrate their functionality. 

int main(void)
{
	Point i{1,0,0}, j{0,1,0}, k{0,0,1}, l{1,1,1};

	std::unordered_set<Triangle> s;
	s.insert(Triangle{i,j,k}); 
	s.insert(Triangle{i,k,j});
	s.insert(Triangle{i,l,k});
	s.insert(Triangle{j,k,l});
	s.insert(Triangle{j,i,k});
	s.insert(Triangle{j,l,k});
	s.insert(Triangle{j,k,i});
	s.insert(Triangle{k,l,j});
	s.insert(Triangle{k,j,i});
	s.insert(Triangle{k,i,j});
	s.insert(Triangle{l,k,i});
	s.insert(Triangle{l,j,i});
	s.insert(Triangle{l,j,k});
	s.insert(Triangle{l,i,j});

	std::cout << s.size() << std::endl;

	return 0;
}

We insert 14 different instances of Triangle which (should) have 14 different hashes.

Running the example produces the following output: 

14

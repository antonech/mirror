string_to_enum function

#include <refltool/string_to_enum.hpp>

The make_string_to_enum_map function returns a std::map, mapping from strings containing the base names of enumerators of enum type T to their values. 

template <typename T >
constexpr std::map<std::string, T> make_string_to_enum_map(void);

Must be an enumeration type

The string_to_enum function returns a std::string containing the base name of the enumerator s of enum type T. Throws std::invalid_argument if s is not a valid enumerator name. 

template <typename T >
T string_to_enum(const std::string& s);

Must be an enumeration type

enum_to_string function

#include <refltool/enum_to_string.hpp>

The make_enum_to_string_map function returns a std::map, mapping from enumerators of enum type T to strings containing the base names of the enumerators. 

template <typename T >
constexpr std::map<T, std::string> make_enum_to_string_map(void);

Must be an enumeration type

The enum_to_string function returns a std::string containing the base name of the enumerator v of enum type T. 

template <typename T >
const std::string& enum_to_string(T v);

Must be an enumeration type

tie_data_members function

#include <refltool/tie_data_members.hpp>

The tie_data_members function returns a std::tuple, containing references to data members inside of the given instance v of a class T. 

template <typename T>
constexpr std::tuple<unspecified&...> tie_data_members(T& v);

The tie_public_data_members function returns a std::tuple, containing references to public data members inside of the given instance v of a class T. 

template <typename T>
constexpr std::tuple<unspecified&...> tie_public_data_members(T& v);

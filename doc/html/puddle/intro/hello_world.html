Hello world

The following code shows a trivial, working example of usage of the PUDDLED operator and the get_base_name and get_scope metaobject operations. 

#include <puddle/reflection.hpp> 
#include <puddle/metaobject_ops.hpp> 
#include <puddle/string.hpp> 
#include <iostream>

namespace Hello { struct World { }; }

int main(void)
{
	using namespace puddle;

	auto mo = PUDDLED(Hello::World);

	auto hello = get_base_name(get_scope(mo));
	auto world = get_base_name(mo);

	auto hello_world = hello + string<',',' '> + world + string<'!'>;

	std::cout << hello_world << std::endl;
	return 0;
}

This header implements the PUDDLED reflection operator.

This header implements the metaobject operations, like get_base_name, etc.

This header implements the compile-time string operations, like concatenation, output to std::ostreams, etc.

See how to implement this example by using the basic reflection facility or the Mirror library.

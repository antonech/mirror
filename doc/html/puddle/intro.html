Hello world

Puddle is a compile-time constexpr metaprogramming-based reflection library built on top of the basic C++ reflection facility and the Mirror reflection library.

While the reflexpr operator and the related library of metaobject operation templates are proposed for standardisation, the Puddle library at this point is not.

Metaobjects

The Puddle library reuses the metaobject types from the Mirror library and uses them as constexpr compile-time values.

Integral constants

#include <puddle/int_const.hpp>

The Puddle library turns the Mirror's integral_constant class templates into the following constexpr variable templates: 

template <typename Int, Int I>
constexpr mirror::int_const<Int, I> int_const;

template <bool B>
constexpr mirror::bool_<B> bool_;

constexpr mirror::true_ true_;
constexpr mirror::false_ false_;

template <char C>
constexpr mirror::char_<C> char_;

template <int I>
constexpr mirror::int_<I> int_;

template <unsigned U>
constexpr mirror::unsigned_<U> unsigned_;

The Mirror's value template alias is implemented as an constexpr function: 

template <typename I>
static inline constexpr
mirror::value<I> value(I)
noexcept;

The Puddle library also provides the basic comparison operators (==, !=, <, >, <=, >=) arithmetic operators (+, -, *, /) and the operator << for output to std::ostream.

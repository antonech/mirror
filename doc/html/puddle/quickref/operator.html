operator PUDDLED

#include <puddle/reflection.hpp>

The metaobjects reflecting various program declarations and other base-level entities can be obtained by invoking the PUDDLED operator.

The result of the PUDDLED operator is an constexpr instance of a specialization of the the mirror::metaobject template wrapping a basic metaobject type: 

#define PUDDLED(X) MIRRORED(X){}

The PUDDLED operator takes the same arguments as the reflexpr and MIRRORED operators.

For example: 

auto meta_int = PUDDLED(int);
auto meta_std = PUDDLED(std);

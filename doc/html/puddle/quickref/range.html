Compile-time ranges

#include <puddle/range.hpp>

The Puddle library turns the Mirror's compile-time range class templates into the following constexpr variable templates: 

template <typename ... P>
constexpr mirror::range<P...> range;

constexpr mirror::empty_range empty_range;

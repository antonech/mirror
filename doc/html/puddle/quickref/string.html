Compile-time strings

#include <puddle/string.hpp>

The Puddle library turns the Mirror's compile-time string class templates into the following constexpr variable templates: 

template <typename Char, Char ... C>
constexpr mirror::basic_string<Char, C...> basic_string;

template <char ... C>
constexpr mirror::string<C...> string;

constexpr mirror::empty_string empty_string;

The Mirror's c_str and string_view template aliases are implemented as an constexpr functions: 

template <typename S>
static inline constexpr
mirror::c_str<S> c_str(S)
noexcept;

template <typename S>
static inline constexpr
mirror::string_view string_view(S)
noexcept;

The Puddle library also provides the basic equality comparison operators (==, !=), concatenation operator (+) and the operator << for output to std::ostream.

Hello world

The following code shows a trivial, working example of usage of the reflexpr operator and the std::meta::get_base_name and std::meta::get_scope metaobject operations. 

#include <reflexpr> 
#include <iostream>
#include <cassert>

namespace Hello { struct World { }; }

int main(void)
{
	using namespace std;

	using meta_HW = reflexpr(Hello::World); 

	assert(is_metaobject_v<meta_HW>);

	cout << meta::get_base_name_v<meta::get_scope_m<meta_HW>>;
	cout << ", ";
	cout << meta::get_base_name_v<meta_HW>;
	cout << "." << std::endl;

	return 0;
}

This header implements the metaobject operation templates.

Reflect the Hello::World struct, the meta_HW type is a metaobject.

See how to implement this example by using the Mirror library or the Puddle library.

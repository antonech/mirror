TagType

Models of meta::TagType reflect the common aspects of elaborated types like classes, unions and enumerations. 

template <Object T> concept bool TagType
 =  Type<T>
 && Scope<T>
 && ScopeMember<T>
 && implementation-defined;

Operations

Operations applicable to metaobjects conforming to the meta::TagType concept: 

// is_enum
template <TagType T>
struct is_enum
 : integral_constant<bool, implementation-defined 
{ };

template <TagType T>
constexpr bool is_enum_v<T> = is_enum<T>::value;

// get_elaborated_type_specifier
template <TagType T>
struct get_elaborated_type_specifier {
	using type = Specifier; 
};

template <TagType T>
using get_elaborated_type_specifier_m
 = typename get_elaborated_type_specifier<T>::type;

true if the meta::TagType T reflects an enumeration type, false otherwise.

true if the meta::TagType T reflects a union, false otherwise.

true if the meta::TagType T reflects a struct, false otherwise.

true if the meta::TagType T reflects a class, false otherwise.

A meta::Specifier reflecting the elaborated type specifier (enum, union, struct or class) of the tag-type reflected by T.

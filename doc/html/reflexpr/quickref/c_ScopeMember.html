ScopeMember

Common aspects of base-level entities declared in a scope (the global scope, a namespace, a class, etc.) are reflected by metaobjects modelling the meta::ScopeMember concept. 

template <Object T> concept bool ScopeMember
 =  implementation-defined;

Operations

Operations applicable to metaobjects conforming to the meta::ScopeMember concept: 

// get_scope
template <ScopeMember T>
struct get_scope {
	using type = Scope; 
};

template <ScopeMember T>
using get_scope_m
 = typename get_scope<T>::type;

A meta::Scope reflecting the scope of the base-level entity reflected by T.

Typed

Models of the meta::Typed concept reflect base-level declarations with a type (for example variables or constants) 

template <Object T> concept bool Typed
 =  implementation-defined;

Operations

Operations applicable to metaobjects conforming to the meta::Typed concept: 

// get_type
template <Typed T>
struct get_type {
	using type = Type; 
};

template <Typed T>
using get_type_m
 = typename get_type<T>::type;

The reflection of the type of the typed declaration reflected by T.

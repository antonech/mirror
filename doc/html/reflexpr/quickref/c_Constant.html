Constant

Compile-time constants (for example enumerators) are reflected by meta::Constant metaobjects. 

template <Object T> concept bool Constant
 =  Typed<T>
 && implementation-defined;

Operations

Operations applicable to metaobjects conforming to the meta::Constant concept: 

// get_constant
template <Constant T>
struct get_constant
 : integral_constant<unspecified , implementation-defined >
{ };

template <Constant T>
constexpr auto get_constant_v<T> = get_constant<T>::value;

Reversible

Metaobjects modelling the meta::Reversible concept can be reverted to the original declaration. This means that for example a meta::Type can be reverted to the base-level type which it reflects by using the meta::get_reflected_type operation. 

template <Object T> concept bool Reversible
 =  implementation-defined;

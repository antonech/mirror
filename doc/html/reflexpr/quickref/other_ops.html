Other metaobject operations

This section documents various operations applicable to multiple different metaobject concepts. 

// is_private
template <Object T>
struct is_private;

template <RecordMember T>
struct is_private<T>
 : integral_constant<bool, implementation-defined 
{ };

template <Inheritance T>
struct is_private<T>
 : integral_constant<bool, implementation-defined 
{ };

template <Object T>
constexpr bool is_private_v<T> = is_private<T>::value;

// get_access_specifier
template <Object T>
struct get_access_specifier;

template <RecordMember T>
struct get_access_specifier<T> {
	using type = Specifier; 
};

template <Inheritance T>
struct get_access_specifier<T> {
	using type = Specifier; 
};

template <Object T>
using get_access_specifier_m
 = typename get_access_specifier<T>::type;

true if the record member reflected by T has private access, false otherwise.

true if the class inheritance reflected by T is private, false otherwise.

true if the record member reflected by T has private access, false otherwise.

true if the class inheritance reflected by T is protected, false otherwise.

true if the record member reflected by T has public access, false otherwise.

true if the class inheritance reflected by T is public, false otherwise.

A meta::Specifier reflecting the access specifier of the record member reflected by T.

A meta::Specifier reflecting the access specifier of the class inheritance reflected by T.

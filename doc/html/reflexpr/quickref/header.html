reflexpr header

The standard <reflexpr> header implements the interface of the metaobject types generated by the compiler.

All reflection-related definitions proposed for standardisation and described in the following sections of this chapter, are nested directly or indirectly in the std namespace.

Note

The namespace-definition opening, 

namespace std {

and closing, 

} // namespace std

statements are assumed, but omitted for the sake of brevity from thisreference.

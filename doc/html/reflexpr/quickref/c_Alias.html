Alias

Some named base-level declarations can have one or several aliases (for example type, template or namespace aliases). These are reflected by metaobjects modelling the meta::Alias concept. All meta::Aliases also conform to the meta::Named concept. 

template <Object T> concept bool Alias
 =  Named<T>
 && implementation-defined;

Operations

Operations applicable to metaobjects conforming to the meta::Alias concept: 

// get_aliased
template <Alias T>
struct get_aliased {
	using type = Named; 
};

template <Alias T>
using get_aliased_m
 = typename get_aliased<T>::type;

The reflection of the original, underlying named declaration aliased by the alias reflected by T.

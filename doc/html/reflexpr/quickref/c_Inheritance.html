Inheritance

Class inheritance declarations are reflected by metaobjects conforming to the meta::Inheritance concept. 

template <Object T> concept bool Inheritance
 =  implementation-defined;

Operations

Operations applicable to metaobjects conforming to the meta::Inheritance concept: 

// get_base_class
template <Inheritance T>
struct get_base_class {
	using type = Record; 
};

template <Inheritance T>
using get_base_class_m
 = typename get_base_class<T>::type;

A meta::Record reflecting the base-class in the inheritance reflected by T.

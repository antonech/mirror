reflexpr operator

The metaobjects reflecting various program declarations and other base-level entities are generated by the compiler on demand by invoking the reflexpr operator.

The result of the reflexpr operator is a special implementation-dependent type conforming to one of the metaobject concepts.

The argument of the reflexpr operator can be one of the following:

  • nothing (no argument) — the result is a meta::GlobalScope.
  • :: — the result is a meta::GlobalScope.
  • specifier — the result is a meta::Specifier.
  • namespace name — the result is a meta::Namespace.
  • namespace alias — the result is a meta::Namespace, meta::Alias.
  • type name — the result is a meta::Type.
  • type alias — the result is a meta::Type, meta::Alias.
  • enum type name — the result is a meta::Enum.
  • union type name — the result is a meta::Record.
  • class name — the result is a meta::Class.

For example: 

using meta_int = reflexpr(int); 
using meta_std = reflexpr(std);

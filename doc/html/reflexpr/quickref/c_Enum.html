Enum

Metaobjects modelling the meta::Enum concept reflect both scoped and non-scoped enumerations. 

template <Object T> concept bool Enum
 =  TagType<T>
 && implementation-defined;

Operations

Operations applicable to metaobjects conforming to the meta::Enum concept: 

// is_scoped_enum
template <Enum T>
struct is_scoped_enum
 : integral_constant<bool, implementation-defined 
{ };

template <Enum T>
constexpr bool is_scoped_enum_v<T> = is_scoped_enum<T>::value;

// get_enumerators
template <Enum T>
struct get_enumerators {
	using type = ObjectSequence; 
};

template <Enum T>
using get_enumerators_m
 = typename get_enumerators<T>::type;

true if the meta::Enum T reflects a scoped (strongly-typed) enumeration, false otherwise.

A meta::ObjectSequence containing meta::EnumMember, meta::Constant metaobjects reflecting the individual enumerators of the enumeration reflected by T.

Metaobject fingerprints

#include <puddle/fingerprint.hpp>

The get_fingerprint function returns a unique, copyable, equality and less-than comparable value for a mirror::metaobject or mirror::none. If two different metaobjects reflect the same base-level declaration then the fingerprint should be the same. 

using fingerprint = unspecified;

fingerprint get_fingerprint(mirror::none);

template <std::meta::Object MO>
fingerprint get_fingerprint(mirror::metaobject<MO>);

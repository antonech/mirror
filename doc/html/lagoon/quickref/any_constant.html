any_constant wrapper

#include <puddle/any_constant.hpp>

The any_constant class is a type-erasure for various integral and enumeration constant types. 

class any_constant
{
public:
	constexpr inline
	any_constant(void)
	noexcept;

	template <typename Src >
	constexpr inline
	any_constant(Src v);

	using value_type = unspecified ;

	constexpr inline
	value_type get(void) const
	noexcept;

	template <typename Dst >
	constexpr inline
	Dst as(void) const;
};

Integral or enum type

As least 64-bit wide unsigned integral type.

Integral or enum type. This must be the same type which was used to initialize the any_constant, otherwise the result is undefined.
